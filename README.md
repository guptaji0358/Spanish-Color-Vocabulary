# #43/100 – Python X Spanish Color Vocabulary

A simple **Spanish Color Vocabulary Library** built using **HTML and CSS**.

This project is part of my **100 Days of Python Code journey**, where I explore how Python connects with web technologies.

---

## 📚 Project Idea

This project displays Spanish color vocabulary with visual color examples.

Each entry contains:
- Spanish color name
- Color image
- Simple vocabulary layout
